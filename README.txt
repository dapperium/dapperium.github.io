DAPPERIUM — Streetwear Bold starter site
Files inside:
- index.html
- assets/css/style.css
- assets/js/script.js
- assets/images/PLACEHOLDER-HERO.svg

How to use:
1. Replace the image 'assets/images/PLACEHOLDER-HERO.svg' with your hero photo named 'hero.jpg' (900x900 recommended) or keep the SVG placeholder.
2. Edit index.html to update text, email, and social links.
3. Open index.html locally to preview.
4. To deploy (FREE): Use Netlify or GitHub Pages.
   - Netlify: Create account -> Sites -> Deploy -> Drag & Drop folder.
   - After deploy, set custom domain (dapperium.com) in Netlify site settings and update DNS records at your domain registrar.
Contact: I can help deploy if you want — reply 'Deploy help'.
